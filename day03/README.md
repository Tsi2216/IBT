# Transaction Log Reader

## Description
This project reads TeleBirr transactions from a text file, calculates the total transaction amount for each customer using a Python dictionary, sorts the customers by total amount in descending order, displays the results, and saves the report to a file.

## Files
- transaction_report.py - Main Python program
- transactions.txt - Input transaction data
- report.txt - Output report generated by the program

## Requirements
- Python 3

## How to Run

1. Open the project folder.
2. Open the terminal.
3. Run:

```bash
python transaction_report.py
